Introduction

Python is an easy to learn, powerful programming language. It has efficient high-level data structures and a simple but effective approach to object-oriented programming. Python's elegant syntax and dynamic typing, together with its interpreted nature, make it an ideal language for scripting and rapid application development in many areas on most platforms.

Python enables programs to be written compactly and readably. Programs written in Python are typically much shorter than equivalent C, C++, or Java programs, for several reasons:

  • the high-level data types allow you to express complex operations in a single statement;
  • statement grouping is done by indentation instead of beginning and ending brackets;
  • no variable or argument declarations are necessary.

Python is extensible: if you know how to program in C it is easy to add a new built-in function or module to the interpreter, either to perform critical operations at maximum speed, or to link Python programs to libraries that may only be available in binary form (such as a vendor-specific graphics library). Once you are really hooked, you can link the Python interpreter into an application written in C and use it as an extension or command language for that application.

Prerequisites

This tutorial is designed for programmers that are new to the Python language, not beginners who are new to programming. Thus, knowledge of coding fundamentals is not recommended, but rather essential.

Getting Started

Python is an interpreted language, which can save you considerable time during program development because no compilation and linking is necessary. The interpreter can be used interactively, which makes it easy to experiment with features of the language, to write throw-away programs, or to test functions during bottom-up program development. It is also a handy desk calculator.

The Python interpreter can be invoked in several ways. On Unix systems, it's typically installed as /usr/local/bin/python3.13, allowing you to start it by typing python3.13 in the shell. On Windows, if Python is installed from the Microsoft Store, the python3.13 command becomes available. Alternatively, the py launcher can be used. To exit the interpreter, you can type quit() or use the end-of-file character (Ctrl-D on Unix, Ctrl-Z on Windows).

The interpreter supports line-editing features like interactive editing, history substitution, and code completion on systems with the GNU Readline library. You can check if command-line editing is supported by typing Ctrl-P at the Python prompt; if it beeps, editing is available.

Python can execute scripts from files, standard input, or command-line arguments. Using python -c 'command' executes the given command, while python -m module runs the specified module as a script. Adding the -i option before a script keeps the interpreter in interactive mode after the script's execution.

In interactive mode, the interpreter prompts with >>> for new commands and ... for continuation lines. For instance:

>>> the_world_is_flat = True >>> if the_world_is_flat: ... print("Be careful not to fall off!") ... Be careful not to fall off!

By default, Python source files are encoded in UTF-8, supporting a wide range of characters. To specify a different encoding, include a special comment at the top of the file:

# -*- coding: encoding -*-
Core Concepts
  1. Data Types

    2. Python has several built-in data types to represent different kinds of values.

    Data Type Example Description
    int 5 Integer number
    float 3.71 Decimal number
    str "Hello Mars!" String (text)
    bool True, False Boolean value
    NoneType None Represents no value
    a = 10 # int b = 3.5 # float c = "Hello" # str d = True # bool e = None # NoneType
  2. Data Structures

    3. These are used to store collections of data.

    List - Ordered, mutable

    fruits = ["apple", "banana", "cherry"] fruits.append("orange") # Add item fruits.pop() # Removes item print(fruits[1]) # Access item

    Tuple - Ordered, immutable

    point = (3, 4) print(point[0])

    Set - Unordered, unique items

    nums = {1, 2, 3, 2} print(nums) # Output: {1, 2, 3}

    Dictionary - Key-value pairs

    person = {"name": "Alice", "age": 30} print(person["name"]) person["age"] = 31 # Update value
  3. Control Flow Statements

    4. Used to make decisions and repeat code.

    Conditional Statements

    x = 10 if x > 5: print("x is greater than 5") elif x == 5: print("x is 5") else: print("x is less than 5")

    Loops

    for i in range(3): # For loop print(i) count = 0 while count < 3: # While loop print(count) count += 1

    Break and Continue

    for i in range(5): if i == 3: break # exits loop print(i) for i in range(5): if i == 0: continue # skips first iteration print(i)
